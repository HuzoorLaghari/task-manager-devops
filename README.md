# 🚀 Task Manager DevOps Project

A containerized Task Manager web application built using **Flask, Docker, and Nginx**, designed with DevOps best practices for scalability, portability, and production-like deployment.

---

## 📌 Features

- ✅ Add and delete tasks
- ✅ Lightweight Flask backend
- ✅ Dockerized application
- ✅ Nginx reverse proxy setup
- ✅ Volume persistence for data
- ✅ Ready for CI/CD integration
- ✅ Production-style architecture
